Overview VRML

  • VRML - Virtual Reality Markup Language
  • VRML biasa dibaca "vermal"
  • VRML cara khusus untuk menampilkan objek 3D melalui World Wide Web
  • VRML memiliki ekstensi file berupa *.wrl
  • VRML bisa dimuat di browser jika browser:
    • adalah browser khusus VRML seperti
      Fraunhofer Institute's CASUS Presenter,
      Sony's Community Place,
      Silicon Graphics' Cosmo Player,
      dan yang lainnya
    • dilengkapi dengan plug-in VRML
  • VRML juga menggunakan prinsip Pemrograman Berbasis Objek

VRML 1.0

Istilah VRML diciptakan/disebutkan oleh Dave Raggett di sebuah paper berjudul "Extending WWW to support Platform Independent Virtual Reality" yang dikumpulkan di acara First World Wide Web Conference pada tahun 1994

Kemudian VRML pertama kali didiskusikan di acara WWW94 VRML BOF yang diadakan oleh Tim Berners-Lee

Pada tahun 1995, Template Graphics Software mendemokan sebuah VRML plug-in untuk peluncuran beta dari Netscape 2.0

VRML 2.0

VRML 2.0 atau VRML 97 adalah spesifikasi VRML yang telah diresmikan oleh ISO (International Standard Organization) pada bulan April 1997 sebagai Bahasa VRML standar dunia

VRML 2.0 lebih sempurna dari VRML 1.0 dengan perbaikan node (Perintah dasar VRML) lama dan pemberian node baru agar lebih interaktif. Selain itu, VRML97 memungkinkan user melakukan desain lingkungan virtual 3D lebih realistis disertai dengan animasi 3D

X3D

X3D adalah standar ISO tanpa royalti untuk merepresentasikan 3D grafik komputer

X3D adalah penerus dari VRML pertahun 2001
